# Rune

Rune is a lightweight local-first CLI for creating, documenting, editing, validating, and running reusable automation scripts.

It helps you keep useful terminal commands and scripts in one place, with a simple module structure, built-in help, diagnostics, and self-update support.

## Why Rune?

Developers often collect useful scripts over time:

- project cleanup commands
- build and run helpers
- deployment shortcuts
- Git workflows
- local automation tools

These scripts usually end up scattered across folders, notes, shell history, or dotfiles.

Rune gives them a simple home.

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/zeidle/rune/main/install.sh | bash
```
